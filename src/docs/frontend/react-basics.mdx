# React 基础教程

## 什么是 React？

React 是一个用于构建用户界面的 JavaScript 库，由 Facebook 开发。它采用组件化的开发模式，让前端开发更加模块化和可维护。

## 核心概念

### 1. 组件 (Components)
React 应用由组件构成，每个组件都是一个独立的、可复用的代码单元。

```jsx
function Welcome(props) {
  return <h1>Hello, {props.name}!</h1>;
}
```

### 2. JSX 语法
JSX 是 JavaScript 的语法扩展，允许我们在 JavaScript 中编写类似 HTML 的代码。

```jsx
const element = <h1>Hello, world!</h1>;
```

### 3. Props（属性）
Props 是从父组件传递给子组件的数据。

```jsx
function App() {
  return <Welcome name="Alice" />;
}
```

### 4. State（状态）
State 是组件内部的数据，当状态改变时，组件会重新渲染。

```jsx
import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);
  
  return (
    <div>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>
        增加
      </button>
    </div>
  );
}
```

## 实战示例

下面是一个简单的待办事项应用示例：

```jsx
import { useState } from 'react';

function TodoApp() {
  const [todos, setTodos] = useState([]);
  const [inputValue, setInputValue] = useState('');

  const addTodo = () => {
    if (inputValue.trim()) {
      setTodos([...todos, inputValue]);
      setInputValue('');
    }
  };

  return (
    <div>
      <h2>待办事项</h2>
      <input 
        value={inputValue}
        onChange={(e) => setInputValue(e.target.value)}
        placeholder="添加新任务"
      />
      <button onClick={addTodo}>添加</button>
      <ul>
        {todos.map((todo, index) => (
          <li key={index}>{todo}</li>
        ))}
      </ul>
    </div>
  );
}
```

## 总结

React 的核心优势：
- **组件化**：代码复用和维护更方便
- **虚拟 DOM**：提高渲染性能
- **单向数据流**：数据流向清晰，易于调试
- **丰富的生态系统**：有大量的第三方库和工具

开始学习 React 吧！这是一个强大而有趣的前端框架。
